## 组件设计

组件分类

- 容器组件
- 通用组件
- 业务组件

---

**容器组件**

- DrawerLayout √
- Layout flex 布局组件
- Tag 组件（支持所有元素标签）

**通用组件**

- Dialog 弹出框组件 √
- Loading 全局加载组件 √
- Nprogress √
- Overlay 遮罩层组件 √
- Toast 提示组件 √
- Lazyload √

**业务组件**

- Image 图片组件（支持点击区域）

- RateList 费率表组件
- Collapse 折叠面板
